# Política de privacidad

Este documento pertenece a **Casa Rural Cádiz de Pruebas**, una web ficticia creada para una actividad formativa y preparada para publicarse con GitHub Pages.

## Responsable

- Nombre comercial: Casa Rural Cádiz de Pruebas
- Correo de contacto ficticio: reservas@casaruralcadiz.test
- Ubicación orientativa: Cádiz, Andalucía, España

## Datos personales

Esta web de demostración no recoge datos personales mediante formularios propios ni almacena información de visitantes en una base de datos.

Los enlaces de contacto abren aplicaciones externas, como el correo electrónico o WhatsApp. Cualquier dato enviado por esas vías quedará sujeto a las políticas de privacidad de dichos servicios.

## Servicios externos

La página utiliza enlaces externos para mostrar imágenes libres y un mapa interactivo de Google Maps. Estos servicios pueden tratar datos técnicos, como la dirección IP o información del navegador, según sus propias condiciones.

## Derechos

Al tratarse de una actividad ficticia, no existe una gestión real de reservas ni de datos personales. Para cualquier consulta sobre el ejemplo, puede escribirse al correo ficticio indicado anteriormente.
